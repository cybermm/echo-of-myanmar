# Echo Of Myanmar Backend Project

- installation
  ```bash
    git clone https://github.com/cybermm/echo-of-myanmar-backend
    composer install
    php artisan key:generate
    cp .env.example .env # add database to .env
    php artisan migrate
    php artisan db:seed # seed for first user
  ```
